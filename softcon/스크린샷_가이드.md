# 포스터용 스크린샷 촬영 가이드

## 필요한 스크린샷 목록

### 1. Multi-vCPU 컬러 출력 (최우선)

**목적**: 4개 vCPU가 동시에 실행되는 것을 색상으로 보여주는 가장 인상적인 데모

**명령어** (4 vCPU - 권장):
```bash
cd /home/seolcu/문서/코드/mini-kvm/kvm-vmm-x86
./kvm-vmm guest/counter.bin guest/multiplication_short.bin guest/hello.bin guest/hctest.bin
```

출력:
- **Magenta (vCPU 0)**: 0-9 카운터
- **Green (vCPU 1)**: 2x 곱셈표 (2x1=2 ~ 2x9=18)
- **Yellow (vCPU 2)**: "Hello, KVM!"
- **Blue (vCPU 3)**: "Hello!"

**촬영 팁**:
- 터미널 크기를 적절히 조정 (너무 작거나 크지 않게)
- 글꼴 크기를 14-16pt로 설정 (가독성 확보)
- 검은 배경 + 컬러 출력이 잘 보이도록
- 4개 vCPU의 출력이 섞여 나오는 것이 명확히 보여야 함
- 스크린샷 후 색상이 잘 나왔는지 확인 (Magenta, Green, Yellow, Blue)

**파일명**: `screenshot_multi_vcpu.png`

### 2. 1K OS 메뉴 화면

**목적**: 9개 프로그램이 포함된 대화형 셸 보여주기

**명령어**:
```bash
cd /home/seolcu/문서/코드/mini-kvm/kvm-vmm-x86
./kvm-vmm --paging os-1k/kernel.bin
```

**촬영 시점**: 메인 메뉴가 표시된 직후
```
Welcome to 1K OS!

Programs:
1. Multiplication Table
2. Counter
3. Echo
4. Fibonacci
5. Prime Numbers
6. Calculator
7. Factorial
8. GCD
9. About 1K OS
0. Exit

Enter program number:
```

**촬영 팁**:
- 전체 메뉴가 깔끔하게 보이도록
- 프롬프트까지 포함
- 색상이 있다면 색상도 보존

**파일명**: `screenshot_1k_os_menu.png`

### 3. 1K OS 프로그램 실행 예시 (선택적)

**목적**: 실제로 프로그램이 동작하는 모습 보여주기

**추천 프로그램**: Fibonacci 또는 Multiplication
```bash
# Fibonacci 실행 예시
printf '4\n0\n' | ./kvm-vmm --paging os-1k/kernel.bin
```

**촬영 내용**:
- 프로그램 선택
- 실행 결과 출력
- 다시 메뉴로 돌아온 화면

**파일명**: `screenshot_1k_os_running.png`

## 스크린샷 촬영 방법

### Linux 스크린샷 도구

**1. GNOME Screenshot (추천)**
```bash
gnome-screenshot -w  # 현재 창만 캡처
gnome-screenshot -a  # 영역 선택 캡처
```

**2. Spectacle (KDE)**
```bash
spectacle
```

**3. Flameshot (강력한 편집 기능)**
```bash
flameshot gui
```

### 터미널 설정 권장사항

**글꼴**:
- 크기: 14-16pt
- 폰트: Monospace 또는 JetBrains Mono

**색상 스킴**:
- 배경: 검은색 (#000000) 또는 진한 회색
- 전경: 흰색 (#FFFFFF)
- vCPU 색상이 잘 보이는 테마 사용

**터미널 크기**:
- 80-120 컬럼
- 24-40 행
- 내용이 잘리지 않도록 조정

## 포스터 배치 권장사항

### 메인 스크린샷 (크게)
- **Multi-vCPU 컬러 출력**: 병렬성을 시각적으로 보여주는 핵심 증거
  - 위치: "개발내용" 섹션 상단
  - 크기: 중간~큰 사이즈
  - 캡션: "4 vCPU 병렬 실행 - 색상으로 구분된 실시간 출력"

### 서브 스크린샷 (작게 또는 중간)
- **1K OS 메뉴**: 완전한 운영체제 구현 증거
  - 위치: "개발내용" 섹션 하단
  - 크기: 작음~중간
  - 캡션: "1K OS - 9개 대화형 프로그램"

### 선택적 스크린샷
- **프로그램 실행 화면**: 공간이 있다면 추가
- **Hello 실행**: 심플함 강조하고 싶다면 추가

## 스크린샷 후처리 (선택적)

### 크롭 (Crop)
- 불필요한 여백 제거
- 중요한 내용만 남기기
- 터미널 테두리는 약간 남겨도 OK

### 리사이즈
- 너비: 최대 1920px
- 고해상도 유지 (DPI 150-300)
- PNG 포맷 사용 (무손실)

### 강조 (선택적)
- 중요한 부분에 화살표나 박스 추가
- 예: "4 vCPU가 동시 실행!" 표시
- 빨간색 또는 노란색 화살표 추천
