## 추가 도구

### Marp CLI 설치 (필요시)
```bash
npm install -g @marp-team/marp-cli
```

### 슬라이드 HTML 생성
```bash
cd /home/seolcu/문서/코드/mini-kvm/softcon
npx @marp-team/marp-cli presentation.md -o presentation.html
```

### 슬라이드 PDF 생성 (백업용)
```bash
npx @marp-team/marp-cli presentation.md -o presentation.pdf
```

### 코드 스니펫 이미지 생성 (Carbon 또는 Silicon)
```bash
# Silicon 설치 (Rust 필요)
cargo install silicon

# 코드 스니펫 이미지 생성 예시
silicon ../kvm-vmm-x86/src/main.c --language c --line-range 100:120 -o code_snippet_irqchip.png
```
